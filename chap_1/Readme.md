# BareMetal Rust Series: Episode 1 🦀

Welcome to the official repository for Episode 1 of the **BareMetal** Rust series! In this project, we write our very first Rust program and master **Cargo**, Rust's powerful build system and package manager.

## 🚀 What's Inside?
This repository contains the exact `hello_cargo` project scaffolded in the video. By exploring this project, you will understand:
* The anatomy of a basic Rust entry point (`main.rs`).
* How to read and configure the `Cargo.toml` control center.
* The difference between development builds and optimized release binaries.
